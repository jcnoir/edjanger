Welcome to edjanger home !

This tool help you to use docker commands in a more compact way. Each docker commands are replaced by a single command which read properties from a configuration file (edjanger.properties).

Design

edjanger is writed in bash shell scripts and commands are alias to be defined in the environment (user profile or system environment).
